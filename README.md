# StudyStack - Next.js + NextAuth + Neon + Prisma + Vercel Blob (TypeScript)

This is a scaffolded starter project for StudyStack — a student document manager.

## Quick start
1. Copy `.env.example` to `.env` and fill in your values.
2. `npm install`
3. `npx prisma generate`
4. `npx prisma db push`
5. `npm run dev`

## Notes
- Replace placeholder upload logic with Vercel Blob signed upload flow.
- Add your Google OAuth credentials and Neon DATABASE_URL in `.env`.
